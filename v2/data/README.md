# Data

Ця папка містить дані для навчання та прогнозування.

## Файли

- matches.csv — історія футбольних матчів
- teams.csv — інформація про команди
- leagues.csv — інформація про ліги
- football_ai.pkl — навчена модель AI

## Призначення

Дані з цієї папки використовуються для:

- навчання моделі;
- перевірки точності;
- створення прогнозів;
- оновлення статистики.
